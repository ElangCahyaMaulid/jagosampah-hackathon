Gamified plastic collection platform to empower Jakarta RT/RW  

**Cara Menjalankan (MVP)**  
1. Backend:  
   ```bash
   cd backend
   composer install
   php artisan migrate --seed
   php artisan serve --port=8000
2. Frontend:
   ```bash
   cd frontend
   npm ci
   npm run dev
